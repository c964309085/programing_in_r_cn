R语言编程概述
--------

说明
1. beamer文件夹中是本课程幻灯片源代码和pdf文件
2. exercise是本课程的练习和数据，练习在html文件中

# 一 学习目标

1. 编写 R 函数
2. 使用 Rmarkdown
3. 编写 R 程序包
4. 使用 git 和 github 进行版本控制

# 二 软件安装

请参考00_software_installation.pdf 先安装好所需软件

# 三 练习

每个讲座结束，将有15分钟左右的练习和答疑时间。 
内容参见： excercises.html
